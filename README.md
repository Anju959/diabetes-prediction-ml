# 🩺 Diabetes Prediction for Pregnant Women using Machine Learning

## 📌 Project Overview

This project predicts whether a pregnant woman is likely to have diabetes based on medical parameters.
It uses Machine Learning algorithms to analyze health data and provide accurate predictions.

## 🎯 Objective

* Early detection of diabetes in pregnant women
* Assist healthcare decision-making
* Reduce health risks for mother and baby

## 🚀 Features

* Easy-to-use prediction system
* Trained Machine Learning model
* Fast and accurate results
* Works with real medical dataset

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 📊 Dataset

* PIMA Indian Diabetes Dataset (`diabetes.csv`)
* Includes features like:

  * Glucose Level
  * Blood Pressure
  * Insulin
  * BMI
  * Age
